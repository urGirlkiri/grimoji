# How To Run Locally

## Web

```bash
flutter run -d chrome
```

> If Images Don't Load try: `flutter run -d chrome --web-browser-flag="--ignore-gpu-blocklist" --web-browser-flag="--enable-webgl"`

## Android

```bash
flutter run -d android
```

## iOS

```bash
flutter run -d ios
```

## Windows

```bash
flutter run -d windows
```

## Linux

```bash
flutter run -d linux
```

## macOS

```bash
flutter run -d macos
```
